Цели проекта:
1. Создание безопасного хранилища для конфиденциальных данных.
2. Обеспечение надежного хранения информации с использованием криптографического шифрования.
3. Реализация удобного и интуитивно понятного пользовательского интерфейса для управления данными.
4. Создание облачной базы данных (Supabase) с возможностью синхронизации и доступа к данным с разных устройств.

Практическая направленность проекта:
Проект имеет прикладное назначение и решает задачу безопасного хранения персональных данных пользователя.
В процессе разработки были реализованы:
- аутентификация пользователей;
- шифрование данных на стороне клиента;
- облачное хранение с разграничением доступа;
- графический интерфейс для просмотра, добавления и управления учетными записями.
